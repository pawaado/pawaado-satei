# パワアド査定ツール

パワプロアドベンチャーズのサクセス終了後に、所持経験点から査定が高くなる能力振りを計算するツールです。

## 変更履歴

### Version 3.0 Phase4
- ツール画面にバージョン表示を追加
- READMEに変更履歴を追加
- 探索状態の圧縮処理を高速化
- 各500点入力時など、経験点が多い条件で計算が長時間化しにくいよう改善
- 計算中にブラウザへ制御を返す頻度を増やし、白画面化を抑制

### Version 3.0 Phase3
- 計算処理をさらに小分け化し、スマホで白画面になりにくいよう改善
- 特殊能力候補の事前除外を追加
- HP別の基本能力候補を整理してから特殊能力探索へ進むよう改善
- 状態数の上限を経験点量に応じて調整
- 結果の残経験点表示は継続

### Version 3.0 Phase2
- 即時エラー表示を追加
- 計算中表示を「計算中」に変更
- ◎の表示を調整
- 計算処理の小分け化を追加

## 注意
小数点以下の査定が不明であることなどから、本ツールの結果が最大値とならない場合があります。また、必殺技、アクションスキル、超特殊能力は割愛しています。あらかじめご了承ください。

「〜攻撃」は、火攻撃、風攻撃、水攻撃のいずれかを指します。

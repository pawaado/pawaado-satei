# パワアド査定ツール

パワプロアドベンチャーズのサクセス終了後に、所持経験点から査定が高くなる能力振りを計算するWebツールです。

## Ver.3.0 Phase2

- 計算中表示を「計算中」に修正
- 経験点1001点以上の即時エラー表示を追加
- 基本能力0以下・上限超えの即時エラー表示を追加
- ◎表示を文字表示に戻し、内側の丸が黒丸に見えないよう修正
- 計算処理を生命力/HP別にまとめて実行し、重くなりにくいよう改善
- 計算中に画面が固まりにくいよう、処理を小分けにして実行

## 注記

小数点以下の査定が不明であることなどから、本ツールの結果が最大値とならない場合があります。また、必殺技、アクションスキル、超特殊能力は割愛しています。

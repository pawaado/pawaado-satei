# パワアド査定ツール

パワプロアドベンチャーズのサクセス終了後に、所持経験点から査定が高くなる能力振りを計算するツールです。

## Phase3 更新内容
- 計算処理をさらに小分け化し、スマホで白画面になりにくいよう改善
- 特殊能力候補の事前除外を追加
- HP別の基本能力候補を整理してから特殊能力探索へ進むよう改善
- 状態数の上限を経験点量に応じて調整
- 結果の残経験点表示は継続

## 注意
小数点以下の査定が不明であることなどから、本ツールの結果が最大値とならない場合があります。また、必殺技、アクションスキル、超特殊能力は割愛しています。あらかじめご了承ください。

「〜攻撃」は、火攻撃、風攻撃、水攻撃のいずれかを指します。


## Version 3.0 Phase5
- 計算中は入力・ボタン操作を無効化
- 計算中表示はそのまま維持
- 探索状態数を見直し、各経験点500程度の条件での計算速度改善を優先
- 計算中に画面が白くなりにくいよう処理をさらに小分け化


## Version 3.0 Phase6
- 基本能力の必要経験点参照列を全面修正
  - 生命力は専用テーブル形式に対応
  - パワー・魔力・器用さ・耐久力・精神力は5種類経験点を正しい列から取得
- 基本能力の取得ごとに経験点を正しく累積減算するよう修正
- 計算後も入力した経験点・基本能力・特殊能力・コツLvを保持
- ツール画面にバージョン表示を追加


## Version 3.0 Phase7

- 計算中の入力保持・操作無効化は継続
- 経験点超過防止・残経験点表示は継続
